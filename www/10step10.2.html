When determining material properties, a distinction is made between materials for which a technical datasheet is available and materials for which no properties have been determined. When a material is recently produced, usually a list of declared properties can be found in the technical datasheet.

Superior

An existing comparable material or material file layout is used and modified according to a full characterization of the properties from experimental testing or obtained from the available technical datasheet.

Advanced

An existing material file is modified to represent the material. The available properties are used to select a comparable material from a hygrothermal material database. Subsequently, the basic properties and moisture-dependent parameters, such as moisture retention, conductivity, and vapor permeability, are scaled or replaced by the known properties from the technical sheets.

Minimum
